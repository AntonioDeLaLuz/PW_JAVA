Ejercicio 9.1

Complete las siguientes oraciones:

  1. Los módulos de programa en JavaScript se llaman funciones.
  2. Una función se invoca mediante el uso de un(a) llamada a función.
  3. Una variable que se conoce sólo dentro de la función en la que se define se llama variable local.
  4. La instrucción return en una función invocada puede usarse para pasar el valor de una expresión de regreso a la función que hizo la llamada.
  5. La palabra clave function indica el comienzo de la defenición de una función.
